# Rede Colmeia Brasil

Plataforma para conexão entre produtores de mel e consumidores, visando a valorização da apicultura brasileira e o fortalecimento do mercado gourmet de produtos apícolas.

## Sobre o Projeto

Este repositório contém o código-fonte da plataforma Rede Colmeia Brasil. A plataforma permite o cadastro de produtores, exibição de produtos gourmet, informações sobre eventos e formação técnica na área de apicultura.

## Como Configurar

1.  Crie um banco de dados MySQL.
2.  Renomeie o arquivo `admin/config.php.template` para `admin/config.php`.
3.  Preencha as credenciais do banco de dados e as configurações de administrador no arquivo `admin/config.php`.
4.  Certifique-se de que a pasta `uploads/` tenha permissões de escrita para o servidor web.
